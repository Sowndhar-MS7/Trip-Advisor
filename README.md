# 🌍 TripAdvisor Clone (HTML & CSS)

A static front-end project that replicates the look and feel of TripAdvisor using **only HTML and CSS**.  
This project focuses on layout, styling, and responsiveness without any JavaScript functionality.

---

## 🚀 Features
- Responsive homepage layout inspired by TripAdvisor
- Navigation bar with logo and links
- Hero section with search bar (non-functional, purely styled)
- Grid-based sections for popular destinations
- Card-style design for hotels, restaurants, and attractions
- Footer with links and social icons

---

## 🛠️ Tech Stack
- **HTML5** → Structure and semantic markup  
- **CSS3** → Styling, Flexbox, and Grid for layout  
- **Responsive Design** → Media queries for mobile, tablet, and desktop
